[한화테크윈 사설 루트 CA 인증서 배포]

1. 압축 해제 후, "HTW_rootca.crt" 파일을 더블 클릭합니다.

2. 본 인증서의 해쉬값은 다음과 같으며, 브라우저에 인증서 설치 시 표시되는 해쉬값과 
   아래의 값이 일치하는지 반드시 비교 확인 후 설치를 진행 하시기 바랍니다.

    - 지문(sha1) : CD28ACD4 F05793EF 43AA524D F30404FD 0B7F3A27

3. 자세한 설치 방법은 별도 제공되는 설치 가이드 문서를 참고해주시기 바랍니다.

※ 반드시 한화테크윈에서 제공하는 루트 CA 인증서만을 사용하시기 바랍니다.




[Hanwha Techwin Private Root CA Certificate]

1. Run "HTW_rootca.crt", after unzip.

2. Before CA installation, check whether the hash value is same as below value.
   Then go to the next step, only when the value is correct.

    - Text(sha1) : CD28ACD4 F05793EF 43AA524D F30404FD 0B7F3A27

3. Refer to installation guide for details.

※ Be sure to use the root CA certificate that Hanwha Techwin provides.
